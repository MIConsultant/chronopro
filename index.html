🔧 Configuration Entra ID
Ces étapes sont à faire une seule fois dans ton tenant M365. Tu as déjà les droits Global Admin — ça prendra environ 10 minutes.
1
Accéder à Entra ID
Ouvre le portail Entra et va dans Applications → App registrations → New registration
https://entra.microsoft.com
2
Créer l'enregistrement
• Name: ChronoPro
• Supported account types: Accounts in this organizational directory only
• Redirect URI: choisir Single-page application (SPA) et coller l'URL complète de ce fichier HTML hébergé sur SharePoint/OneDrive
3
Copier les identifiants
Dans la page Overview de l'app, note le Application (client) ID et le Directory (tenant) ID
4
Ajouter les permissions API
API permissions → Add → Microsoft Graph → Delegated:
• Files.ReadWrite (accès OneDrive)
• User.Read (profil utilisateur)
5
Saisir les identifiants dans l'app
Retourne dans l'onglet ⚙️ Config et colle le Client ID et le Tenant ID.
